## 用户相关

### 登录
#### 简要描述：
- 用户登录接口
#### 请求url：
- /api/user/login
#### 请求方式
- POST
#### 参数：
参数名|必选|类型|说明
-|-|-|-|
username|是|String|用户名
password|是|String|密码
#### 返回格式
``` json
    {
        "code":1000,
        "msg":"sssss"
    } 
```
#### 返回参数说明
参数名|必选|类型|说明
-|-|-|-|
msg|是|String|错误说明
